#Machine Learning 

- Le Machine Learning est la brique de base de l'IA, la chaîne dans le vélo, le moteur dans la voiture, le rouage de l'automate.
- Le Machine Learning a une tâche précise à accomplir : prédire.
- Le Machine Learning utilise des algorithmes pour développer des modèles prédictifs à partir de jeux de données (datasets, en anglais).

##2 types de modélisation : 

###1 - Modélisation Statique (MS) : 
Démarche d'analyse et d'interprétation de la dynamique entre les variables.
S'appuie sur des hypothèse et modèles maths
(ex : Quel est le profil des utilisateurs qui s'abonnent ?)

###2 - Modélisation Prédictive (MP) : 
Approche Machine Learning (ML)
(ex : Comment prédire si un nouvel utilisateur va s’abonner ?)

2 attendues du MP : 
- Prédiction de qualité
- Capacité à extrapoler -> robustesse du modèle

Data trop lourdes (type audio, images, vidéos) = Deep Learning.
